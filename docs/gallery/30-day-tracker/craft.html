The first set is your warm-up build; the last working set is what counts. If you complete every prescribed rep with clean form, add the next week's load. If you miss, repeat the week.

Rules of the road

Form before load. Sleep before supplements. Track every working set. If a weight feels wrong, drop a plate and live to lift next week.
